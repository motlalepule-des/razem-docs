## Platform Overview

Razem is a South African financial services platform enabling businesses and individuals to:

- Send and receive EFT and real-time payments
- Manage beneficiaries and payment schedules
- Access account information and transaction history via API
- Build custom payment integrations using the Razem REST API

| Component | Description | Audience |
|:---|:---|:---|
| **My Razem Account** | Web portal for account management and payments | Business users |
| **Razem API** | REST API for programmatic access | Developers |
| **Razem Docs** | This documentation site | Everyone |
