# My Razem Account

The **My Razem Account** portal (`my-razem-account`) is the customer-facing web application that provides individuals and businesses with full control over their Razem accounts.

## Key Features

- **Dashboard** — Real-time account overview, balance, and recent transactions
- **Payments** — Send money via EFT or Real-Time Payments
- **Beneficiaries** — Manage saved payment recipients
- **Statements** — Download monthly account statements
- **Notifications** — Configure payment and account alerts
- **Security** — Two-factor authentication, device management, and session control
- **Developer Access** — Generate and manage API keys

## Accessing the Portal

The My Razem Account portal is accessible at:

- **Production:** [account.razem.co.za](https://account.razem.co.za)
- **Staging:** [account-staging.razem.co.za](https://account-staging.razem.co.za)

## Technology Stack

The My Razem Account portal is built with modern web technologies to ensure a fast, responsive, and secure user experience. It communicates with the Razem API for all data operations.
