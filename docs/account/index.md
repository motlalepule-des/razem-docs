# Business Guide

Everything you need to manage your Razem account and process payments through the **My Razem Account** portal.

This section is for **account holders, business owners, and finance teams** who use Razem to send and receive payments.

{: .note }
Are you a developer building an integration? See the [Developer Documentation](/razem-docs/docs/technical) for the API reference, code examples, and architecture guides.

---

## What You Can Do

| Feature | Description |
|:---|:---|
| **Account Setup** | Register, complete KYC verification, and configure your profile |
| **Payments** | Send EFT, real-time payments, and scheduled or bulk payments |
| **Receive Money** | View incoming payments and share your account details |
| **Beneficiaries** | Save and organise payment recipients |
| **Statements** | Download transaction history in PDF or CSV format |
| **Security** | Set up 2FA, manage trusted devices, and control account access |
